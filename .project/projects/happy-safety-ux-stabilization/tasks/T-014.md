# Task: Harden command and skill message parsing

## Description

Replace brittle regex-only command wrapper handling with a tolerant parser and fixtures for long skill markdown, malformed wrappers, embedded closing tags, escaped XML, multiple wrapper blocks, and markdown in command args.

## Acceptance Criteria

- [ ] Locate current parser and message rendering path.
- [ ] Add fixtures for pathological command and skill wrapper content.
- [ ] Parse command name, args, wrapper body, and malformed cases deterministically.
- [ ] Treat long internal wrapper bodies as internal implementation detail by default.
- [ ] Preserve ordinary user-authored text.

## Definition of Done
