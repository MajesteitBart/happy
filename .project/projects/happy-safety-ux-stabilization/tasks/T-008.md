# Task: Gate app sends on server capabilities

## Description

Teach the app and/or CLI to check capabilities before enabling message send paths that require v3 message APIs. Incompatible self-host servers should produce structured upgrade guidance.

## Acceptance Criteria

- [ ] Fetch and cache capabilities during server connection setup.
- [ ] Block unsupported message-send paths with a clear compatibility error.
- [ ] Add app/client tests for missing capability and supported capability paths.
