# Task: Enforce strict local secret file permissions

## Description

Ensure Happy home directories, credentials, session cache, daemon state, lock files, and key material are created with explicit strict permissions and repaired or warned on startup when unsafe.

## Acceptance Criteria

- [ ] Centralize safe directory and secret-file write helpers.
- [ ] Create Happy home directories with `0700` where supported.
- [ ] Write credential/session/key/state files with `0600` where supported.
- [ ] Warn or repair insecure existing permissions.
- [ ] Add cross-platform tests or guarded behavior for unsupported file mode platforms.
