# Task: Classify abort and process exit states

## Description

Fix the user-facing ambiguity where mobile aborts can surface as repeated `Process exited unexpectedly` states. User abort, expected cancellation, runner crash, delivered-but-not-consumed, and unexpected exit should produce different states and recovery guidance.
