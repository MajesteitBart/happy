# Task: Add handoff regression coverage

## Description

Build tests or scripted integration fixtures for the highest-risk handoff paths: desktop to mobile, mobile back to desktop, remote session creation, daemon/tmux-style detached sessions, and switch behavior with active shell/monitor processes.

## Acceptance Criteria

- [ ] Add a fixture or test for desktop-started session becoming visible on mobile.
- [ ] Add a fixture or test for mobile-sent message reconciling back to desktop context.
- [ ] Add a fixture or test for remote-created session appearing in local/daemon session lists.
- [ ] Add a fixture or documented gap for detached/tmux-style multi-session behavior.
- [ ] Verify active subprocess behavior is preserved or explicitly marked cancelled.
