# Task: Classify abort and process-exit lifecycle states

## Context

Upstream #1343 reports aborts turning into repeated `Process exited unexpectedly` states. Related upstream #982 reports mobile messages crashing a Claude local session with SIGTERM. The user-facing state needs to distinguish user aborts from launcher crashes and delivery failures.

- [ ] Add focused tests or a documented smoke trace.
